# Mediun
A blog for posting articles built on top of Facebook Rich text editing Using 
  - React/Redux
  - NodeJS
  - GraphQL

### Installation
Installing client dependencies.

```sh
$ cd Mediun/client
$ npm install
```
installing server dependencies
```sh
$ cd Mediun/server
$ npm install
```
### Run Mediun
```sh
$ cd Mediun
$ cd npm run dev
```
### see the Client in action
visit http://localhost:8080
